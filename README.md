# Dynamic Quiz App

A responsive quiz builder built with HTML, CSS, and JavaScript. Users can create custom multiple-choice questions, start a quiz, answer each question, and view the final score.

## Features

- Add unlimited custom quiz questions
- Four answer options per question
- Dropdown-based correct answer selection to avoid typing mistakes
- Question validation before adding to the quiz
- Added question preview list
- Edit questions after adding them
- Remove individual questions or clear the full quiz
- Start quiz only when questions are available
- One-question-at-a-time quiz flow
- 30-second timer for each quiz question
- Required answer selection before submitting
- Final score with percentage
- Restart quiz and create-new-quiz actions
- Responsive layout for desktop and mobile

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How To Run

Open `index.html` in your browser.

No build tools or package installation are required.

## How It Works

1. Enter a question.
2. Add four possible answers.
3. Select the correct answer from the dropdown.
4. Click `Add Question`.
5. Add more questions if needed.
6. Edit or remove questions from the preview list if needed.
7. Click `Start Quiz`.
8. Select answers and submit each question before the timer ends.
9. View your final score.
